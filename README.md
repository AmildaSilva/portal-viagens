✈️ Portal de Viagens
Site de viagens desenvolvido com Next.js para apresentar destinos turísticos do mundo. O projeto usa App Router src/app, a abordagem mais atual do Next.js  para roteamento e renderização.

🔗 Demonstração
Acesse o projeto em: http://localhost:3000
Repositório: https://github.com/AmildaSilva/portal-viagens.git


📦 Scripts disponíveis
Comando
npm run dev
npm run build


🚀 Tecnologias
Tecnologia	Uso
Next.js 16	Framework React com App Router e SSR/SSG
React 18	Biblioteca para UI
TypeScript	Tipagem estática
CSS Modules	Estilização com escopo local
React Icons	Ícones do Header/Footer
📁 Estrutura do Projeto
O projeto utiliza App Router, a estrutura recomendada desde o Next.js 


✅ Funcionalidades
Listagem de destinos na home
Página de detalhes com rota dinâmica /destinos/[id]
Layout com Header e Footer
Estilização responsiva com CSS Modules
Imagens otimizadas com next/image [x]
